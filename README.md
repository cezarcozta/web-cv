### Getting Started

>Yarn and NPM **REQUIRED**.

```bash
  # Clone this repository.
  $ git clone https://github.com/cezarcozta/web-cv.git


  # Access repository cloned folder
  $ cd web-cv

  # Install all dependencies
  $ yarn

  # Run development server
  $ yarn start

  # Check Browser
  $ http://localhost:30000
```
